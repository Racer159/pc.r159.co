What makes a good CPU:

Speed of Cores

allows a process to execute faster

Number of Cores

allows more processes to run at once

Age of CPU

newer CPUs have more features and are generally more efficient

What makes core "fast"

A "fast" core is made from a combination of clocking quickly (measured in GHz) and executing instructions in as few clock cycles as possible (measured in Instructions per Cycle).

How many cores do I need?

Modern games (2022) benefit from a minimum of 4 cores

(ideally with simultaneous multi-threading also known as hyperthreading)

How new of a processor do I need?

With the 12th generation of Intel, the processor wars between AMD and Intel have heated up significantly, so, ideally within the last 2 years. There are some unicorns from previous generations if you are really strapped for cash, however, you will need to do your research on these.

How do I compare CPUs easily?

User Benchmark

For a good rough cut, this is a decent option. It allows you to select CPUs across generations and compare their effective speed against one another. Do keep in mind that their data is user provided so can be skewed by thermal issues, windows updates, or other things users don't consider before benchmarking.
